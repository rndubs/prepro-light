# Sample Mesh Files

This directory contains sample mesh files for testing the Prepro Light extension.

## Files

- **cube.vtp** - A simple cube mesh in VTK PolyData format
- **sphere.vtp** - A simple icosphere mesh in VTK PolyData format

## Usage

To test the extension:
1. Press F5 to launch the extension in debug mode
2. In the Extension Development Host window, open any of the sample files
3. The mesh should be displayed in the 3D viewer

## File Formats

- `.vtp` - VTK PolyData (XML format)
- `.vtu` - VTK Unstructured Grid (XML format) - coming soon
- `.vti` - VTK ImageData (XML format) - coming soon
- `.stl` - Stereolithography format - coming soon
- `.obj` - Wavefront OBJ format - coming soon
